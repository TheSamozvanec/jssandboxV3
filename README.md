<h1>Описание работы с offline-версией редактора</h1>

<p>
    Offline-версия требует запуска через локальный HTTP-сервер. Это связано с тем, что браузеры блокируют загрузку модулей и выполнение некоторых скриптов при открытии через <code>file://</code>. Используйте любой локальный сервер (например, <code>npx serve</code>, <code>python -m http.server</code> или приложение типа «Файловый менеджер+» на телефоне).
</p>

<p>
    Offline-версия полностью соответствует приложению, которое размещено в сети интернет по адресу <a href="https://semov777.com/sandbox">https://semov777.com/sandbox</a>. Постоянно обновляемое руководство пользователя также находится в сети интернет: <a href="https://semov777.com/?section=sandbox">https://semov777.com/?section=sandbox</a>.
</p>

<h2>Установка в телефоне</h2>

<p>
    Для установки на телефон необходимо скачать ZIP-архив из данного репозитория и распаковать его в удобную папку на телефоне. Рекомендую использовать приложение <b>«Файловый менеджер+»</b> по следующим причинам:
</p>

<ul>
    <li>Приложение поддерживает ZIP-архивы.</li>
    <li>Приложение способно запускать HTML в браузере через <code>http://127.0.0.1:14942/</code>.</li>
    <li>Именно этим приложением я тестировал работу на телефоне.</li>
    <li>CORS-политика Open API (<a href="https://poligon.semov777.com">https://poligon.semov777.com</a>) позволяет работать с <code>http://127.0.0.1:14942</code>.</li>
</ul>

<p>
    Для скачивания архива нажмите зелёную кнопку <b>CODE</b> и выберите пункт <b>Download ZIP</b>.
</p>

<p>
    После скачивания архив обычно попадает в папку «Загрузки». Вы можете переместить архив в удобную папку и извлечь содержимое. С помощью приложения «Файловый менеджер+» необходимо запустить <code>index.html</code> с помощью любого браузера (из проверенных — Яндекс и Chrome; к сожалению, Kiwi не вызывает виртуальную клавиатуру).
</p>

<p>
    Модули песочницы, размещённые на официальном сайте, также содержатся в архиве. Их можно найти в папке <code>assets/modules/</code>.
</p>

<p>
    В офлайн-версии вы можете использовать собственные модули, а также делать произвольные запросы через <b>fetch</b>. В веб-версии ограничения установлены сервером, но здесь их уже нет.
</p>

<h2>Установка на desktop</h2>

<p>
    Если вам понадобилось установить офлайн-версию на desktop, вы можете также скачать ZIP-архив или клонировать репозиторий.
</p>

<p>
    Для запуска откройте командную строку, перейдите в папку с файлом <code>index.html</code> и выполните команду:
</p>

<pre><code>npx serve .</code></pre>

<p>
    Затем перейдите по предложенной ссылке.
</p>

<p>
    <b>Важно:</b> если просто открыть файл <code>index.html</code> в браузере, скрипты и CSS не будут загружены, и приложение не будет работать.
</p>
